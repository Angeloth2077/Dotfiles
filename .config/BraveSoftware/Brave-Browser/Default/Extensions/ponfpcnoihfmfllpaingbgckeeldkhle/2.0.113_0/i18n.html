Internationalization

You can help translate Enhancer for YouTube™ to one of the supported locales without writing a single line of code. Here are the steps you need to follow:

  1. Select your language
  2. Click on the right side of the string(s) you want to translate to enter the edit mode
    Note: The “English (USA)” is always up-to-date and must be used as a reference for all the translations
  3. When you have finished translating the string(s) click the “Generate code” button
  4. Grab the generated code and follow the last steps to save it to a file and to send it

Important: If you decide to translate more than just a few strings, to avoid you unnecessary work you should contact the developer (webmastermrfdev.com) to ask if someone as already started the same translation. It is also highly recommended to regularly click the “Generate code” button to grab the code and save it temporarily because if your browser updates the extension while you are translating strings this page will be closed without any warning and all your work will be lost.

Except where otherwise noted you must use the infinitive form.

Thank you very much for your contribution! Here are the last steps you need to follow:

  1. Somewhere in your documents, create a text file named messages.txt
  2. Copy the generated code and paste it into the file you've just created then save the file
  3. Log in to your mailbox to compose a new message
  4. Copy the text you see next to the message icon above and use it as subject for your message
  5. Attach the file messages.txt and feel free to comment your contribution in your message
  6. Send your message to webmastermrfdev.com
